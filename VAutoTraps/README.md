# VAutoTraps

Trap/kill-streak systems and trap command hub.

## Commands
- `.trap` help hub
- `.trap set/remove/list/...`
- `.spawnchest ...`
- `.streak ...`

## Config
- `Configuration/killstreak_trap_config.toml`

## Prefab Conversion
Use `VAuto.Core.PrefabGuidConverter` to convert prefab names to GUIDs and back.
