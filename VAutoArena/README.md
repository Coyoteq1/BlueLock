# VAutoArena

Standalone arena plugin with territory-based glow borders and arena commands.

## Commands
- `.arena` help hub
- `.arena enter` / `.ae`
- `.arena exit` / `.ax`
- `.arena glow spawn|clear|test|validate`
- `.arena territory show|reload`

## Config
- `config/VAuto.Arena/arena_territory.toml`
- `config/VAuto.Arena/arena_glow_prefabs.toml`

## Prefab Conversion
Use `VAuto.Core.PrefabGuidConverter` to convert prefab names to GUIDs and back.
